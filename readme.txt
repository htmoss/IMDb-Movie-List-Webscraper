---USE---

1. Find an IMDb public list, some examples are "100 Best Action Movies" (https://www.imdb.com/list/ls009668579/) and "100 Best Rom-Coms" (www.https://imdb.com/list/ls058479560/)
2. Enter the full URL into the string "URL" on line 6
3. Enter the name of what you want your CSV to be titled on line 8
4. Run the code and see the CSV you have created! (see imdb_scrape.csv for finished example)
